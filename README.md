# TreeEdit

纯Qml实现的树结构编辑器。

使用Controls2 中的 ListView 和 ListModel模拟实现。

|功能|进度|
|--|--|
|树结构的缩进|完成|
|展开、折叠|完成|
|添加|完成|
|删除|完成|
|重命名|完成|
|搜索|完成|
|导入|计划中|
|导出|计划中|
|节点属性编辑|计划中|
